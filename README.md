# Ilsa Khan Interior — Luxury Portfolio Website

This zip contains a complete static website inspired by the uploaded luxury editorial interior-design references.

## Files
- `index.html` — Home page
- `spaces.html` — Portfolio / project grid
- `project-palm-residence.html` — Project detail template
- `approach.html` — Approach page
- `story.html` — Studio / story page
- `affiliates.html` — Collaborations page
- `contact.html` — Contact page
- `styles.css` — Full responsive styling
- `script.js` — Menu, scroll reveal, simple form interactions
- `assets/` — Original SVG placeholder images. Replace these with your real project/interior images.
- `tempo-prompt.txt` — One-shot prompt for Tempo/no-code AI website builders.

## How to use
1. Open `index.html` in a browser to preview.
2. Replace images inside `/assets` with your real images using the same filenames, or update the image paths in HTML.
3. Upload the code to your hosting, or paste the prompt in `tempo-prompt.txt` into Tempo.

## Important
The design is original and branded for Ilsa Khan Interior. It is inspired by the reference screenshots but does not use the original site logo, copy, or final assets.
